Learn to code JavaScript Operators for beginners

An operator performs some operation on single or multiple operands (data value) and produces a result. With operators we can also concatenate strings and much more.

JavaScript Operators

The assignment operator (=) assigns a value to a variable:

var x = 10;

The addition operator (+) adds numbers:

  • const x = 5;
  • const y = 2;
  • const z = x + y;

The multiplication operator (*) multiplies numbers:

  • const x = 5;
  • const y = 2;
  • const z = x * y;
